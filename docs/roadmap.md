# Roadmap

This document outlines the current state and planned future development of **DesignXpress AI Story Video Studio**.

## Current State (v1.0)

### Completed Major Features
- Full-stack cinematic AI video editor (Next.js + Express)
- Advanced multi-track FFmpeg rendering with real video clips
- BullMQ export queue with progress tracking and history
- Real-time collaboration (cursors, scene sync, chat)
- Templates Marketplace with publish/load
- Project Version History
- Playable video assets in the Studio preview
- PayPal checkout experience (sandbox)
- Comprehensive local Windows development experience
- Production Docker setup + deployment guides

## Near-Term Goals (Next 3–6 Months)

### High Priority
- **Real Video Preview Scrubbing** — Smooth playback and scrubbing of actual video clips in the timeline
- **Export Queue Dashboard** — Dedicated UI for managing queued and completed renders
- **Public Project Viewer** — Fully functional page for viewing shared projects
- **Better Usage Tracking** — Track AI generations and export minutes per user

### Medium Priority
- Full ElevenLabs + Stability AI integration with cost tracking
- Per-scene comments and annotations (real-time)
- Media library improvements (tagging, collections, search)
- Team workspaces and role-based permissions

## Long-Term Vision

- **Cloud-Native Version** — Hosted version with scalable rendering infrastructure
- **Advanced AI Features**:
  - AI video effects and transitions
  - Automatic scene detection and editing suggestions
  - Voice cloning and emotional control
- **Marketplace Expansion** — Monetization for template creators
- **Mobile Companion App** — Review and approve projects on mobile
- **Enterprise Features** — SSO, audit logs, custom branding

## How to Influence the Roadmap

We welcome community input! The best ways to shape future development are:

1. Open a GitHub issue with the `feature-request` label
2. Join discussions in the repository
3. Contribute code for features you're passionate about (see [Contributing Guide](./contributing.md))

---

**Note**: This roadmap is aspirational and subject to change based on user feedback and development capacity.
